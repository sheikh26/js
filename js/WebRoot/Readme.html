Java Script

  1. Javascript functions are written in <script type="text/javascript"> </script> tag
  2. <script > tag can be wriiten anywhere in HTML but <HEAD> tag is prefered to write java script functions because head postion is load in browser first.
  3. Function is always start with 'function' key word
  4. Variables are definbed by 'var' keyword ex 'var msg =0'
  5. Variable data type depened on its stored value ex.
    1. var msg = 1 ; //integer
    2. var msg ="Galaxy" //string
    3. var msg = 1.5 //float
    4. etc.
  6. String can be representaed by single quote (') or double quote (") ex.
    1. var org = "Galaxy " Or var org = 'Galaxy'
